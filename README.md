pour ouvrir le frontend :
cd frontend
npm install
npm install tailwindcss @tailwindcss/vite
npm run dev 


pour ouvrir le backend :
cd backend
npm install
npm init
npm i cors
npm i express
npm i body-parser 
npm i dotenv 
npm i mongoose 
npm i --save-dev nodemon
nodemon index.js 
npm install jsonwebtoken
npm run dev
